More on QQ Plots

QQ Plots allow us to evaluate whether two samples come from the same distribution. Here are some examples comparing a simulated dataset to the standard normal distribution.

Code 
normal <- rnorm(1000)

par(mfrow = c(1, 2))
hist(normal, col = "gray", border = "white", breaks = 50)
qqnorm(normal)
abline(0, 1, col="red")

If a distribution is skewed left, the end points will fall below the 45-degree line.

Code 
skew.left <- rchisq(1000, 2) * -1

par(mfrow = c(1, 2))
hist(skew.left, col = "gray", border = "white", breaks = 50)
qqnorm(skew.left)
abline(0, 1, col="red")

If a distribution is skewerd right, the end points will fall above the 45-degree line.

Code 
skew.right <- rchisq(1000, 2)

par(mfrow = c(1, 2))
hist(skew.right, col = "gray", border = "white", breaks = 50)
qqnorm(skew.right)
abline(0, 1, col="red")

If a distribution has light tails, this means there are fewer small and large values than what we would expect in the standard normal distribution.

Code 
light.tail <- runif(1000, min = -3, max = 3)

par(mfrow = c(1, 2))
hist(light.tail, col = "gray", border = "white", breaks = 50)
qqnorm(light.tail)
abline(0, 1, col="red")

If a distribution has heavy tails, this means there are more extreme values (high and low) than what we would expect in the standard normal distribution.

Code 
heavy.tail <- rt(1000, 4)

par(mfrow = c(1, 2))
hist(heavy.tail, col = "gray", border = "white", breaks = 50)
qqnorm(heavy.tail)
abline(0, 1, col="red")

Example: Broockman, Kalla, and Aronow save Political Science

In 2015, political scientists David Broockman, Joshua Kalla, and Peter Aronow documented some fishy-looking data from an article published in Science by a political science graduate student: Irregularities in LaCour (2014)

One key concern of Broockman et al was that LaCour’s data looked strikingly similar to a previous survey sample, even though LaCour had claimed the studies to be original random samples. How did they show this? They compared the distributions of the LaCour study data with the previous data from the 2012 Cooperative Campaign Analysis Project (CCAP).

You can check out the link for their full report, but here’s just a quick example of what they found. Note: this is not the exact data or procedures in Broockman et al.

The histograms of the LaCour studies look practically identical to each other and to the CCAP data, even though they are supposed to come from different random samples.

Code 
par(mfrow=c(2,2))
hist(lacour$therm1,
     main = "LaCour 1",
     breaks = 5)
hist(lacour$therm2,
     main = "LaCour 2",
     breaks = 5)
hist(lacour$therm3,
     main = "LaCour 3",
     breaks = 5)
hist(ccap$gaytherm,
     main = "CCAP",
     breaks = 5)

QQ Plots comparing the first LaCour sample to the CCAP sample.

Code 
ccap$gaytherm2 = ifelse(is.na(ccap$gaytherm), 50, ccap$gaytherm) # if missing, set to 50
par(mfrow=c(1,2))
qqplot(lacour$therm1, ccap$gaytherm, main = "Without NAs",
       xlab = 'LaCour 1', ylab = 'CCAP')
abline(0, 1, col="red")
qqplot(lacour$therm1, ccap$gaytherm2, main = "NAs set to 50",
       xlab = 'LaCour 1', ylab = 'CCAP')
abline(0, 1, col="red")

QQ Plots comparing the second and third LaCour samples to simulated data from the CCAP study with added noise from a normal distribution.

Code 
ccap$gaytherm3 = ccap$gaytherm2 + rnorm(length(ccap$gaytherm2), mean = 0, sd = 5)
ccap$gaytherm3[ccap$gaytherm3 < 0] <- 0
ccap$gaytherm3[ccap$gaytherm3 > 100] <- 100
ccap$gaytherm4 = ccap$gaytherm3 + rnorm(length(ccap$gaytherm3), mean = 0, sd = 5)
ccap$gaytherm4[ccap$gaytherm4 < 0] <- 0
ccap$gaytherm4[ccap$gaytherm4 > 100] <- 100

par(mfrow=c(1,2))
qqplot(lacour$therm2, ccap$gaytherm3, main = "LaCour 2 vs. CCAP Simulated",
       xlab = 'LaCour 2')
abline(0, 1, col="red")
qqplot(lacour$therm3, ccap$gaytherm4, main = "LaCour 3 vs. CCAP Simulated")
abline(0, 1, col="red")

Try it Yourself: ANES (2020) Feeling Thermometers

library(haven)
load('clean_anes.rda') # load data

Feeling thermometer question: We’d also like to get your feelings about some groups in American society. When I read the name of a group, we’d like you to rate it with what we call a feeling thermometer. Ratings between 50 degrees-100 degrees mean that you feel favorably and warm toward the group; ratings between 0 and 50 degrees mean that you don’t feel favorably towards the group and that you don’t care too much for that group. If you don’t feel particularly warm or cold toward a group you would rate them at 50 degrees. If we come to a group you don’t know much about, just tell me and we’ll move on to the next one.

This subset of the 2020 ANES dataset contains feeling thermometers towards the following groups:
